This game is about 50% completion. The Sprite were created with PiskelApp.
Animation run on canvas. The project started as a revisit of the asteroids
game, but found other inspiration.

controls:

a---- (move left), d---- (move right)
left arrow---- (move left) right arrow---- (move right)
shift---- (move in opposite direction at same speed)
space bar----(shoot)

goal:

Destroy all 9 ships without getting hit by a laser blast.

recommended browsers:
Firefox
Chrome
